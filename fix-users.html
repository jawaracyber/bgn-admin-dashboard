Fix Users

Portal Data Nasional - Satsus Pronas

⚠️ Perhatian: Tool ini digunakan untuk memperbaiki user yang tidak memiliki entry di tabel user_roles. Anda harus login sebagai SUPER_USER untuk menggunakan tool ini.
Fungsi Tool Ini: 1. Memeriksa semua user di auth.users
2. Mengidentifikasi user yang tidak memiliki entry di user_roles
3. Membuat entry user_roles untuk user yang bermasalah
4. Menampilkan log detail dari proses perbaikan
